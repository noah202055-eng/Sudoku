Sudoku Puzzle Generator
Noah Jin
My code generates a 9x9 sudoku board that is fully solved. I used array lists and 2 d arrays for storage and organization and filled the board by starting with an empty grid and placing a number from 1-9 in each cell and making sure that sudoku rules are not violated. After a valid sudoku is made, it is printed.
Sudoku.java is the file with the code.
There is a second file with a more in depth description called Noah Jin - Project_ Sudoku Puzzle Generator [50 points].pdf and a video explanation of my code.
